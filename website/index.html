Features

Everything in the desktop app — from live rates to chest capacity and Steam pricing.

Live

Big XP/hour (held steady between game saves), gold/hour, session totals, and current map. Per-hero level and XP/hour breakdown, an XP-updated counter, scrollable history of every XP change, session reset, and an idle warning after two minutes.

Inventory

Owned items resolved against bundled catalogs, grouped by type with composition stats. Search, filter, and sort by grade, type, location, tradable, and in-use. Steam price and value columns, location breakdown (inventory, stash, trading, equipped), and graceful handling of unknown items after game updates.

Chests

Unopened chest slots from your save: common (gray), stage boss (blue), and act boss (red). Capacity from base slots plus purchased rune nodes and settings bonuses. Full badges, progress bars, and remaining-slot counts for each category.

Market

Pick a currency and refresh Steam Market prices. Background job runs on save load and backs off on rate limits until complete.

Mini overlay

Frameless, always-on-top readout with XP and gold rates, current map, priced inventory total, and pricing status. Toggle from Mini in the tab bar.

Box tracker overlay

Track rare boss box drops with 12-minute timers and community ideal-stage routes. Mark drops, clear timers, and see which stage you're on — open from the tab bar or Chests tab.

Settings

Edit config.json from the app: save path, poll interval, rolling window, currency, cube XP tracking, CSV history logging, and always-on-top. Warns before changes that reset your session.

  • savePath — path to SaveFile_Live.es3
  • es3Password — ES3 decryption password
  • pollIntervalSeconds — how often to re-read the save
  • rollingWindowMinutes — window for XP/hour
  • currency — ISO code for Steam prices (USD, EUR, BRL, …)

How it works

1

Reads your local save

The app watches SaveFile_Live.es3 on disk and decrypts it locally. No connection to game servers.

2

Never modifies anything

Read-only access. Your save file is never written to, and the companion does not inject or automate gameplay.

3

Windows for now

Official builds are Windows NSIS installers. macOS and Linux are not packaged yet.

Fan-made tool. Not affiliated with Tesseract Studio. Does not modify your save or connect to game servers.

FAQ

Is it safe? Does it modify my save?
No. TBH Companion only reads your encrypted save file. It never writes to the save, never talks to game servers, and never automates the game.
Where is my save file?
By default the app looks for %USERPROFILE%\AppData\LocalLow\TesseractStudio\TBH\SaveFile_Live.es3. You can override the path in Settings or config.json.
Decryption stopped working after a game update
The developer may have rotated the ES3 password. Update es3Password in Settings (or config.json) and restart the app. See the project docs on save format for details.
How does Steam pricing work? Will I hit rate limits?
Prices are fetched from the public Steam Market API when you open the Market tab or load a save. The app backs off automatically if Steam rate-limits requests and retries until done. Pricing is best-effort — some items may show as unknown.
Is this an official TBH / Task Bar Hero product?
No. TBH Companion is a fan-made, unofficial tool. It is not affiliated with or endorsed by Tesseract Studio or the TBH: Task Bar Hero developers.
Do I need Windows?
Yes, for the pre-built installer. Releases are Windows NSIS installers. You can build from source on other platforms, but packaged builds are Windows-only for now.
How are chest capacities calculated?
Each chest type (common, stage boss, act boss) starts at 5 base slots. Purchased capacity runes in RuneSaveData add +1 slot per node (Rune of Containment, Vault, and Infinity). Common chests can also get bonus slots from settings. The Chests tab shows the full breakdown from your save.
How do I clear cache or reset app data?
Quit the app, then delete its user-data folder. Development builds use %APPDATA%\tbh-companion; the NSIS installer uses %APPDATA%\TBH Companion. That removes config, logs, and cached prices. Your game save is untouched.
Why does XP/hour look different from in-game?
The companion computes rates from save snapshots over a rolling window (default 5 minutes). XP/hour is held steady between periodic game saves instead of decaying, so short dips in the game UI may not match.
